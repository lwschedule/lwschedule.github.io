# Code Consolidation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use compose:subagent (recommended) or compose:execute to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Eliminate ~1,190 lines of duplicated code across 4 consolidation areas without losing features or making code convoluted.

**Architecture:** Extract shared logic into `common.js` as parameterized utilities, then reduce each page to a thin wrapper that calls the shared code with page-specific DOM targets.

**Tech Stack:** HTML, CSS, vanilla JS (no frameworks)

## Global Constraints

- Every commit bumps the z version (x.y.**z**)
- No feature loss — every button, toggle, and behavior must work identically
- Keep code human-readable — no over-abstraction
- Static PWA — no build step, no bundler
- Each page must still work as a standalone HTML file
